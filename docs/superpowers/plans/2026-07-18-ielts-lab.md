# IELTS Lab Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone IELTS practice platform for sentence patterns, four-sentence templates, short exercises, and progress saving.

**Architecture:** A static app with `index.html` for structure, `styles.css` for the full responsive interface, `app.js` for interaction and local persistence, and `tests/site.test.mjs` for content/behavior smoke checks.

**Tech Stack:** HTML, CSS, vanilla JavaScript, Node built-in test runner via a plain script.

## Global Constraints

- No dependency installation.
- The first screen must be the study tool, not a landing page.
- IELTS content must be independent from the TCM/GPA personal platform.
- User-facing content should include practical sentence patterns and four-sentence templates.
- Progress and writing/speaking notes should save locally in the browser.
